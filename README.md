# Faith That Shows Up - 31-Day Devotional Tracker

A highly polished, responsive Next.js single-page application built for couples to track their spiritual intimacy progress over a 31-day walk.

## Features

- **Dual Partner Progress**: Independent check-offs for "His Walk" and "Her Walk" with micro-animations.
- **Spiritual Unity**: Tracks days where both partners have completed their devotional readings.
- **Interactive Overview**: Displays the F-A-I-T-H strategy structure, progress rings, and scripture quotes.
- **Weekly Sub-navigation**: Weeks 1-5 sub-tabs containing scriptures (linked directly to BibleGateway) and reflection prompts.
- **Auto-saved Weekly Journals**: Separate workspaces for His and Her journal logs that auto-save draft state to browser localStorage.
- **Clean Aesthetic**: Modern calm design with warm cream backgrounds, slate texts, earthy sage greens, and deep teals.

## Technical Stack

- **Framework**: Next.js (App Router, React 19)
- **Styling**: Tailwind CSS v4, Lucide React
- **Persistence**: Safe client-side localStorage synchronization hook

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application in the browser.
